DesignatedRuleForObject class

This is used during tracing to indicate that an object should be processed using a particular rule, rather than the one that might ordinarily be found for it. This normally only makes sense if the object is uniquely referenced, and is a more or less internal collection. See ListRuleEssential for an example. It knows how to return its object and how to filter.

class DesignatedRuleForObject {
 Function rulePredicate;
 final target;

 DesignatedRuleForObject(this.target, this.rulePredicate);

 possibleRules(List rules) => rules.where(rulePredicate).toList();
}
